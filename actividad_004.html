Ejercicio A - Verificación de identidad

Hacé un programa que realice verificación de identidad. El programa deberá pedir una clave al usuario y este tendrá tres oportunidades para poner la clave correcta. a) Si el usuario pone la clave incorrecta 3 veces, se deberá imprimir “Acceso denegado”. b) Si puso la clave mal, pero todavía le quedan oportunidades, imprimir “Vuelva a ingresar la clave”. c) Si se ingresó la clave correcta, imprimir “Acceso otorgado”.
